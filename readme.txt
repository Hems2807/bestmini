📦 FindIt — Lost & Found Web App
A simple and smart Lost & Found item management system built with HTML, CSS & JavaScript.
Users can add lost or found items, upload images, search items, and claim found belongings by uploading proof (receipt/screenshot).
✅ No backend — data stored using localStorage
✅ Modern responsive UI
✅ Works on mobile & desktop
✅ Image preview before upload
✅ Proof upload for claiming items
🎯 Features
Feature	Description
👤 Login (demo)	Enter any username & password
➕ Add Items	Add lost or found items with image & details
🖼️ Image Preview	Shows uploaded item photo
🔎 Search Bar	Live
